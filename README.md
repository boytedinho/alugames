## Overview

AluGames allows users to rent and return boardgames. The dashboard displays available games with the option to rent or return a game.

## Getting Started

1. Clone or download the repository to your local machine.
2. Open the `index.html` file in your preferred web browser.

## Features ğŸ®

### 1. Dashboard Display

- **Game Listings:** View a list of available boardgames.
- **Renting and Returning:** Click buttons to rent or return a game.
- **Visual Status:** Quickly identify rented games with visual indicators.

### 2. Dynamic Interaction

- **Renting Games:** Increases the count of rented games.
- **Returning Games:** Decreases the count of rented games.
- **Confirmation Dialog:** Confirm before returning a rented game.

## How to Use ğŸ•¹ï¸

1. Open the application in your browser.
2. Browse the available boardgames in the dashboard.
3. Click the "Alugar" (Rent) button to rent a game.
4. For rented games, click the "Devolver" (Return) button to return the game.
5. Confirm the return in the dialog box.
6. Keep track of the total rented games displayed.

## Counting Rented Games

The application dynamically counts and displays the total number of rented games.
